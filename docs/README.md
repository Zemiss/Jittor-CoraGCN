# Documentation

这里放对开源读者有用的项目文档。临时计划、草稿和个人执行记录请放在本地 `Plan/` 或 `plan/` 目录中，这些目录不会进入版本控制。

## Index

- [Challenge](challenge.md)：赛题背景、输入输出、数据字段和提交要求。
- [Workflow](workflow.md)：从环境准备、训练、推理到提交校验的完整流程。
- [Project Completion](project_completion.md)：当前实现结果、验证记录和复现说明。
- [Open Source Guidelines](open_source_guidelines.md)：计图比赛项目开源整理规范。

## Recommended Reading Order

1. 先读 [Challenge](challenge.md)，确认任务目标和提交格式。
2. 再读 [Workflow](workflow.md)，按步骤复现训练和打包。
3. 需要核对当前结果时读 [Project Completion](project_completion.md)。
4. 维护仓库结构时参考 [Open Source Guidelines](open_source_guidelines.md)。
